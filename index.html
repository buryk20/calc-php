Калькулятор подбора АКБ для ИБП W

Путем использования инструмента подбора аккумуляторных батарей для устройства бесперебойного питания, возможно провести приближенный расчет необходимой емкости аккумулятора для данного источника непрерывного питания.

РАСЧЕТ ЕМКОСТИ АККУМУЛЯТОРНОЙ БАТАРЕИ ДЛЯ ИСТОЧНИКА БЕСПЕРЕБОЙНОГО ПИТАНИЯ

Нагрузка, W:

Суммарная нагрузка, которая подается на ИБП. Можно посмотреть в тех документации приборов, которые подключены к ИБП

Введите значение

КПД инвертора %:

КПД инвертора можно посмотреть в паспорте ИБП, если вы не знаете это значение, то ставьте по умолчанию 80.

Номинальное напряжение АКБ, V:

Номинальное напряжение аккумуляторной батареи.

Введите значение

Время работы:, ч:

Время которое должен проработать ИБП

Введите значение

РЕЗУЛЬТАТ:

Минимальная ёмкость АКБ, ч = 12
